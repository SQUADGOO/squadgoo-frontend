# SQUADGOO Admin Panel

A comprehensive admin dashboard for managing the SQUADGOO platform.

## Features

- **Dashboard**: Analytics, charts, and overview statistics
- **User Management**: Manage jobseekers and recruiters
- **KYC Verification**: Review and approve identity documents
- **Job Offers**: Manage job postings and marketplace listings
- **Wallet & Payments**: Handle transactions, withdrawals, and escrow
- **Dispute Center**: Resolve conflicts between users
- **Support**: Live chat, tickets, and FAQ management
- **Content Management**: Manage FAQs, tips, and announcements
- **Staff Management**: Manage admin staff and permissions
- **Settings**: Configure fees, legal documents, and system settings
- **Security**: Activity logs, sessions, and 2FA management

## Tech Stack

- React 18
- Redux Toolkit (State Management)
- Tailwind CSS (Styling)
- Chart.js (Charts)
- Font Awesome (Icons)
- React Router (Navigation)
